# Dotfiles

A repository to hold various shared configurations.

### Prerequisites
- Git
- npm
- Homebrew
- GNU Stow
- An Installed Nerd Font

1. [Clone this repo]
1. [Init Stow]
1. [Install omz]
1. [Install neovim]
